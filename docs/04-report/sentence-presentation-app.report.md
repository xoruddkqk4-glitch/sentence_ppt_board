## 1. 요약

`sentence-presentation-app`는 영어 교사가 지문을 입력하면 문장 단위로 분석하고, 교실 뒤쪽 학생도 볼 수 있는 큰 프리젠테이션 화면으로 보여주는 정적 웹 앱이다.

이번 PDCA 사이클에서는 입력, 문장 분석, 분석 결과 수정, 드래그 앤 드랍 보정, 문장 성분 색상 표시, 종요소 단계별 공개, TXT 저장/불러오기까지 포함한 수업용 흐름을 구현하고 문서화했다.

최종 분석 기준 Match Rate는 94%로 목표 기준 90%를 넘었다.

## 3. 완료 항목

### 3.1 입력과 문장 분리

- [x] 교사가 영어 지문을 입력하거나 붙여넣을 수 있다.
- [x] 지문을 문장 단위로 분리한다.
- [x] 샘플 문장으로 바로 앱을 확인할 수 있다.
- [x] 구두점과 apostrophe를 표시 화면에 보존한다.

### 3.2 문장 성분 분석과 수정

- [x] 주어, 동사, 목적어, 보어를 주요소로 구분한다.
- [x] 형용사어, 부사어를 종요소로 구분한다.
- [x] 교사가 성분 역할을 select로 수정할 수 있다.
- [x] 교사가 성분을 드래그 앤 드랍으로 재배치할 수 있다.
- [x] 선택한 단어만 분리할 수 있다.
- [x] 선택한 여러 단어는 하나의 묶음으로 분리할 수 있다.
- [x] 겹치게 드래그한 성분은 병합할 수 있다.
- [x] 형용사어가 수식하는 원문 단어를 복수 선택할 수 있다.

### 3.3 프리젠테이션 화면

- [x] 주요소를 한 줄에 크게 표시한다.
- [x] 주요소 라벨은 본문에 붙이지 않고 색상으로 구분한다.
- [x] 색상 범례를 화면 위쪽에 표시한다.
- [x] 종요소는 하나의 성분이 한 줄에 표시된다.
- [x] 주요소 먼저 보여주고, 클릭 또는 다음 동작마다 종요소를 하나씩 누적 공개한다.
- [x] 종요소 anchor는 문장 맨 앞, 단어 사이, 문장 맨 뒤 위치에 맞춘다.
- [x] 종요소가 화면 밖으로 나가지 않도록 글자 크기와 위치를 보정한다.
- [x] 형용사어는 수식 대상 괄호와 화살표로 표시한다.
- [x] 긴 문장과 긴 종요소에 맞춰 글자 크기를 동적으로 보정한다.

### 3.4 조작과 저장

- [x] 이전, 다음, 첫 문장, 마지막 문장 버튼을 제공한다.
- [x] 마지막 문장에서는 마지막 문장 버튼을 `이전 단계`로 바꾼다.
- [x] 오른쪽/왼쪽 화살표는 공개 단계 이동에 사용한다.
- [x] 위/아래 화살표는 문장 단위 이동에 사용한다.
- [x] 화면 클릭은 다음 버튼과 같은 역할을 한다.
- [x] 분석 결과를 인덱스 포함 TXT 파일로 저장할 수 있다.
- [x] 저장한 TXT 파일을 불러오면 문장과 분석 결과를 복원한다.
- [x] 지원 브라우저에서는 저장 위치 선택 창을 사용한다.

## 7. 배운 점

### Keep

- 교사가 직접 수정할 수 있는 구조가 문법 분석 오류를 보완한다.
- 발표 화면에서 주요소와 종요소를 단계적으로 공개하면 수업 설명 흐름을 만들기 쉽다.
- TXT 저장/불러오기는 서버 없이도 수업 준비물을 재사용하게 해준다.

### Problem

- 종요소 anchor와 연결선은 화면 폭, 단어 길이, 종요소 개수에 따라 쉽게 겹칠 수 있다.
- apostrophe 같은 작은 구두점은 토큰화 과정에서 사라지기 쉽다.
- 구현이 빠르게 바뀌면 설계 문서가 뒤처져 다음 수정의 기준이 흐려질 수 있다.

### Try

- Playwright 기반 스크린샷 QA를 도입한다.
- 긴 문장과 긴 종요소 예제를 회귀 테스트 세트로 유지한다.
- 문법 분석 정확도를 높이되, 교사 수정 흐름은 계속 핵심 기능으로 유지한다.

## 8. 남은 리스크

- 규칙 기반 분석이므로 복잡한 관계절, 병렬 구조, 삽입구에서는 성분 오분류가 생길 수 있다.
- 브라우저 크기와 전자칠판 해상도에 따라 종요소 박스 배치가 다르게 보일 수 있다.
- 파일 저장 위치 선택은 브라우저 지원 여부에 따라 다운로드 방식으로 대체될 수 있다.

## 9. 다음 단계

PDCA 완료 보고서가 작성되었으므로 다음 단계는 archive이다.

```text
$pdca archive sentence-presentation-app
```

archive 전에는 실제 브라우저에서 다음 항목을 한 번 더 확인하는 것이 좋다.

- `Let's`, `today's` 같은 apostrophe 포함 문장
- 문장 맨 뒤에 붙는 부사구 종요소
- 여러 종요소가 같은 anchor 근처에 붙는 문장
- 형용사어가 주요소와 종요소를 각각 수식하는 문장
- 저장한 TXT 파일을 다시 불러오는 흐름
