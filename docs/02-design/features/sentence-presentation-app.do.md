# sentence-presentation-app - Do 단계 구현 기록

> 작성일: 2026-05-30 | 상태: 완료

## 1. 구현 범위

설계 문서 기준으로 Starter 레벨 정적 웹 앱을 구현했다.

구현 파일:

- `index.html`
- `css/style.css`
- `js/main.js`

## 2. 구현된 화면

- 입력 화면: 영어 지문 입력, 샘플 지문 입력, 초기화, 발표 준비
- 수정 화면: 문장별 성분 분석 결과 확인, 주요소/종요소 줄 표시, 역할 수정, 드래그앤드롭 이동
- 발표 화면: 주요소 윗줄, 종요소 아랫줄, 이전/다음 이동, 수정/편집 이동

## 3. 구현된 주요 기능

- 영어 지문 문장 분리
- 규칙 기반 문장 성분 초안 생성
- 주요소: 주어, 동사, 보어, 목적어
- 종요소: 형용사어, 부사어
- 발표 화면 주요소 한 줄 고정
- 발표 화면 주요소 라벨 제거 및 색상 구분
- 발표 화면 상단 색상 범례
- 종요소 성분별 개별 줄 표시
- 종요소의 문장 내 위치 기준 정렬
- 성분 묶음 내부 줄바꿈 방지
- 수정 화면에서 드래그앤드롭으로 성분 이동
- 성분 역할 변경
- 밝은 테마 / 어두운 테마 전환
- 발표 글자 크기 자동 조정
- 발표 화면 키보드 조작

## 4. 설계 반영 사항

- `Pretendard`, `system-ui`, `sans-serif` 폰트 스택 적용
- 발표 화면 글자 크기에 `clamp()`와 텍스트 길이 기반 자동 축소 적용
- 발표 화면에서는 드래그앤드롭 비활성화
- 수정 화면에서만 드래그 핸들 표시
- 성분 묶음은 `white-space: nowrap`으로 내부 줄바꿈 방지

## 5. 알려진 한계

- 문장 성분 분석은 규칙 기반 초안이므로 복잡한 문장의 분석이 틀릴 수 있다.
- 첫 구현의 드래그앤드롭은 브라우저 기본 HTML Drag and Drop API 중심이다.
- 터치 전자칠판에서의 세부 조작감은 추가 확인이 필요하다.
