# Sentence PPT Board

영어 수업 중 교사가 영어 지문을 입력하면 문장을 자동으로 나누고, 문장 성분을 큰 발표 화면으로 보여주는 교실용 정적 웹앱입니다. 교실 뒤쪽 학생도 볼 수 있도록 주요소를 크게 표시하고, 종요소는 문장 안에서 들어가는 위치를 점과 선으로 연결해 보여줍니다.

## 주요 기능

- 영어 지문을 문장 단위로 자동 분리
- 주어, 동사, 목적어, 보어를 주요소로 분류해 한 줄에 표시
- 형용사어, 부사어를 종요소로 분류해 아래 줄에 표시
- 주요소는 라벨 대신 색으로 구분하고, 색상 범례를 화면 상단에 표시
- 종요소가 들어가는 위치를 문장 앞, 단어 사이, 문장 뒤 슬롯에 점으로 표시
- 교사가 분석 결과를 드래그 앤 드롭으로 수정
- 구 단위 성분을 원하는 단어 묶음으로 분리
- 겹치게 드래그해 분리된 성분을 다시 병합
- 긴 주요소와 종요소를 화면 안에 들어오도록 동적으로 축소
- 분석 결과를 TXT 파일로 저장하고 다시 불러오기
- 마지막 발표 화면에서 첫화면 이동과 화면 끄기 지원

## 사용 방법

1. `index.html`을 브라우저에서 엽니다.
2. 입력 화면에 영어 지문을 붙여넣습니다.
3. `발표 준비`를 눌러 문장 분석 결과를 만듭니다.
4. 분석 결과 수정 화면에서 성분 역할, 위치, 분리, 병합을 조정합니다.
5. `발표 시작`을 눌러 학생용 발표 화면으로 전환합니다.
6. `이전`, `다음` 버튼이나 키보드 방향키로 문장을 넘깁니다.

## TXT 저장과 불러오기

`TXT 저장`을 누르면 현재 문장과 분석 결과가 `.txt` 파일로 저장됩니다. 지원되는 브라우저에서는 저장 위치를 직접 선택할 수 있고, 지원되지 않는 경우 기본 다운로드 방식으로 저장됩니다.

저장 파일에는 사람이 읽을 수 있는 문장 번호, 성분 번호, 역할, 줄 구분, 단어 인덱스가 포함됩니다. 파일 하단에는 앱이 다시 불러올 수 있는 JSON 데이터가 함께 들어갑니다.

`TXT 불러오기`를 누르고 저장했던 TXT 파일을 선택하면 문장, 분석 결과, 성분 위치가 복원됩니다.

## 발표 화면 조작

- `다음`: 다음 문장으로 이동
- `이전`: 이전 문장으로 이동
- `수정`: 분석 결과 수정 화면으로 이동
- `TXT 저장`: 현재 분석 결과 저장
- `첫화면`: 입력 화면으로 이동
- `화면 끄기`: 검은 화면 오버레이 표시
- `화면 켜기`: 검은 화면 해제

키보드 조작:

- `→` 또는 `Space`: 다음 문장
- `←`: 이전 문장
- `Home`: 첫 문장
- `End`: 마지막 문장
- `Esc`: 수정 화면으로 이동

## 브라우저 참고 사항

TXT 저장 위치 선택은 브라우저의 File System Access API를 사용합니다. Chrome과 Edge 계열 브라우저에서는 저장 위치 선택 창이 뜰 수 있습니다. 지원하지 않는 브라우저에서는 일반 다운로드 방식으로 저장됩니다.
